Control librtlsdr (https://github.com/librtlsdr/librtlsdr) where controls aren't available in the app in use

Uses UDP to existing functionality

Written using freepascal / lazarus IDE, can be compiled by downloading the IDE (https://www.lazarus-ide.org/) or github actions

The UDP component available (lnet) is non-blocking - this causes some issues

I am planning to update librtlsdr to tag the value being sent to this app
